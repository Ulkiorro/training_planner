# Training Planner-План тренировок

## Автор
Артём Яценко

## Описание программы
Training Planner — это desktop-приложение с графическим интерфейсом для планирования и отслеживания тренировок. Программа позволяет добавлять записи о тренировках (дата, тип, длительность), фильтровать их по типу и дате, а также сохранять данные в JSON-файл.

## Как запустить
```bash
python training_planner.py
